o operador not nega a sentença atribuída e se acoplado com o if pode ser de muita utilidade em diversos algoritmos:

## algoritmo sem not
```js
if (possuiCNH) {
	PodeDirigir = true;
} else{
PodeDirigir = false;
}
```

## algoritmo com not
```js
if (!possuiCNH) {
	PodeDirigir = false;
} else{
PodeDirigir = true;
}
```
